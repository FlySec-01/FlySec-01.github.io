提取安装包

获取包列表：adb shell pm list packages
获取包地址：adb shell pm path <packagename>
导出包：adb pull <path>

解包分析

打开雷电模拟器安装后运行，发现提示app对设备进行了验证判断，因此要对其进行解包分析绕过限制。

使用压缩包的格式打开apk，提取其中的dex 文件

使用dex工具将dex转为jar

通过 jadx gui 反编译 jar，通过搜索关键字寻找相关代码
